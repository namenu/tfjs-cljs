# tfjs-cljs

A ClojureScript wrapper library for [TensorFlow.js](https://js.tensorflow.org/).

## Usage

Add `tfjs-cljs` as a dependency to your `project.clj`.

```
[tfjs-cljs "0.1.0"]
```

Then add the following to your `ns`.

```
(:require [tfjs-cljs :as tf])
```
